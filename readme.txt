برای دیپلوی یک کانترکت ساده روی شبکه Base با استفاده از Remix، می‌توانی مراحل زیر را دنبال کنی:

### مرحله 1: نوشتن کد کانترکت

1. **وارد Remix شو**: به آدرس [Remix IDE](https://remix.ethereum.org) برو.
2. **ایجاد فایل جدید**: در پنل چپ، روی "File Explorers" کلیک کن و یک فایل جدید با پسوند .sol بساز. مثلاً SimpleStorage.sol.
